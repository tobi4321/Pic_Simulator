Ausführliche Beschreibung

class Controller

This class is the heart of this Simulator. It is the connection between all classes. Each interaction by a user on the GUI is executed in the controller class. Here are objects of processor, memory, timer, interrupt and watchdog.

◆ loadMnemonicFromTable()

void Controller.loadMnemonicFromTable ( )
protected

Method to load the program code from the code view table. If labels are found in the 5th column they are added. Otherwise the blank mnemonic code from the 6th column is checked for EQUs. If it contains EQUs the line is added, otherwise the line is added with a space as prefix. Then the mnemonic code string is loaded into the mnemonic editor.
