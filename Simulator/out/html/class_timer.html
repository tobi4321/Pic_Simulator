class Timer

This class is used for implementation of a simple Timer

If the Timer is active, the Timer Register is incremented each tick by specific Time

T0SE 81h Bit 4 - 1 high to low T0CS 81h Bit 5 - 1 is ra4/t0cki - 0 is clkout T0IF 0bh Bit 2 - 1 is set by overflow of register 01h T0IE 0bh Bit 5 - 1 allows Timer to set a interrupt

PSA 81h Bit 3 - 0 is prescaler is active for timer PS0 81h Bit 0 PS1 81h Bit 1 PS2 81h Bit 2
